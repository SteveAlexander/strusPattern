Detailed Description

Options to stear regular expression automaton.

Remarks
Available options depend on implementation
Note
For the hyperscan backend, the following options are available: "CASELESS", "DOTALL", "MULTILINE", "ALLOWEMPTY", "UCP"
The options HS_FLAG_UTF8 and HS_FLAG_SOM_LEFTMOST are set implicitely always
